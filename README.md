# 🗿 Susisesi's Totem Pack (Open Source / Açık Kaynak)

[EN] This is an open-source custom totem texture for Minecraft. You are free to use, modify, and integrate this design into your own resource packs.
[TR] Bu, Minecraft için hazırlanmış açık kaynaklı özel bir totem dokusudur. Tasarımı özgürce kullanabilir, değiştirebilir ve kendi paketlerinize ekleyebilirsiniz.

---

## Features / Özellikler
* **Original Design:** Created from scratch. / **Özgün Tasarım:** Sıfırdan oluşturuldu.
* **Open Source:** Direct access to source files. / **Açık Kaynak:** Kaynak dosyalara direkt erişim.
* **Compatibility:** Works with latest versions. / **Uyumluluk:** En güncel sürümlerle çalışır.

## Installation / Kurulum
1. Download the `.zip` file. / `.zip` dosyasını indirin.
2. Place it into your `resourcepacks` folder. / `resourcepacks` klasörüne atın.
3. Activate it in-game. / Oyun içinden aktif edin.
